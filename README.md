## Tech Stack  

Svelte, Vite, SCSS, Typescript, CSS


## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/RAPrograms/Tic-Tac-Toe-Web-Client
~~~

Go to the project directory  

~~~bash  
  cd Tic-Tac-Toe-Web-Client
~~~

Install dependencies  

~~~bash  
npm install
~~~

Create build files

~~~bash  
npm run build
~~~

Grab build files from 

~~~bash  
./dist
~~~
